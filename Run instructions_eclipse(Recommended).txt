1. Download the zip file "weather-api-final" and extract all files.

2. Open Eclipse and click file in up left, select "open projects from the file system"

3. Load the "weather-api" folder from "weather-api-final" (Allow public and private access for Eclipse if asked)

4. Navigate to src/main/java in weather-api project

5. Open "WeatherApiApplication.java" and run it.(Make sure localhost:8081 is available)

      if localhost:8081 is not available kill the port by executing the following commands:

          Open Command Prompt or PowerShell as Administrator and run the following command:
                --> netstat -ano | findstr :8081
          This will display the PID of the process using port 8081.
          Then, stop the process using the taskkill command:
                --> taskkill /PID <PID> /F
6. Open Web browser and got the url: http://localhost:8081/weather/forecast/32.85,-96.86?date=2025-04-01

   The url format is as follows: http://localhost:8081/weather/forecast/latitude,longitude?date=Date

   we can change latitude, longitude and Date as per locations and date.
